# Portfolio Website

A modern, responsive portfolio site for a fullstack web developer. Built with **HTML**, **Tailwind CSS**, and **vanilla JavaScript** — no build step, no framework, just clean code.

> 🤖 This project was crafted entirely through a conversation with an AI on Telegram.

## ✨ Features

- 🎨 **Glassmorphism UI** with animated gradient orbs
- ⌨️ **Typing animation** in the hero section
- 🪟 **Scroll reveal** effects on every section
- 📱 **Fully responsive** — mobile menu, adaptive grid
- 🎯 **Sticky navbar** with active section highlighting
- 💼 **Project showcase** with hover interactions
- 📈 **Animated skill bars**
- 📬 **Contact form** with submit animation
- 🌗 Dark theme by default

## 🛠️ Tech Stack

| Layer    | Tools                          |
| -------- | ------------------------------ |
| Markup   | HTML5                          |
| Styling  | Tailwind CSS (CDN) + custom CSS |
| Behavior | Vanilla JavaScript (ES6)       |
| Fonts    | Inter, JetBrains Mono          |

## 📂 Structure

```
portfolio/
├── index.html      # Main page (single-page layout)
├── style.css       # Custom styles, animations, glassmorphism
├── script.js       # Typing effect, scroll reveal, form, navbar
└── .gitignore
```

## 🚀 Getting Started

No installation needed. Just open the file:

```bash
# Option 1: open directly
open index.html

# Option 2: serve locally
python3 -m http.server 8000
# then visit http://localhost:8000
```

## 🎨 Customization

- **Name, role, content** → edit `index.html`
- **Colors, animations** → tweak `tailwind.config` block at top of `index.html`
- **Custom styles** → `style.css`
- **Interactions** → `script.js`

## 📄 Sections

1. **Hero** — name, typing intro, CTAs, social links
2. **About** — bio, location, stats
3. **Skills** — tech categories + proficiency bars
4. **Projects** — 4 case study cards
5. **Experience** — timeline of roles
6. **Testimonials** — client quotes
7. **Contact** — form + email/social links
